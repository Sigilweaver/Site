Rust quickstart

use opentfraw::RawFileReader;

let raw = RawFileReader::open_path("sample.raw")?;
println!("{} -- {} scans", raw.device_family.display_name(), raw.num_scans);

let mut file = std::fs::File::open("sample.raw")?;
for scan_num in 1..=raw.num_scans {
    let peaks = raw.read_scan_peaks(&mut file, scan_num)?;
}

Python quickstart

The python/ sub-crate wraps the Rust library via PyO3. Build with maturin: 

cd python && maturin develop --release

Format coverage

File format versions 8, 47, 57, 60, 62, 63, 64, and 66 — covering every Thermo instrument line from the LCQ Classic (mid-1990s) through the Orbitrap Astral (2023).

Parsed components: file header, audit tags, sample info, run header, scan event hierarchy and filters, scan index, peak data (profile and centroid packets), error log, instrument log.
