## Development

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
cd site
npm install
```

### Local Development

```bash
npm run dev
```

Starts a development server at `http://localhost:25808` with hot module replacement.

### Build

```bash
npm run build
```

Generates optimized static files in the `dist/` directory.

`npm run build` also regenerates:
- `public/sitemap-core.xml` for marketing pages
- `public/sitemap.xml` as a sitemap index that points to:
  - `/sitemap-core.xml`
  - `/docs/sitemap.xml` (generated by Docusaurus)
  - `/openyxdb/sitemap.xml` (generated by Docusaurus)

You can regenerate only the sitemap files with:

```bash
npm run sitemap
```

### Preview Production Build

```bash
npm run preview
```

Serves the production build locally for testing.

## Adding New Pages

1. Create the HTML file in the root directory:

```html
<!-- newpage.html -->
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>New Page - Sigilweaver</title>
  <link rel="stylesheet" href="/src/styles/main.css">
</head>
<body>
  <!-- Content -->
</body>
</html>
```

2. Update navigation links in other pages as needed.

3. Add to `vite.config.ts` if using multi-page setup:

```typescript
export default defineConfig({
  build: {
    rollupOptions: {
      input: {
        main: resolve(__dirname, 'index.html'),
        newpage: resolve(__dirname, 'newpage.html'),
      },
    },
  },
});
```
