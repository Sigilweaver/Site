Understanding Asset Names

Variants:

  • Full Application: Complete desktop app with bundled backend (Python/Polars runtime included)
  • Lightweight Native (Tauri): Smaller native app using system webview, requires separate backend server
  • Client Only: Frontend-only build that connects to a separate backend server
  • Web Version: Static files for self-hosting the web UI

Formats:

  • .exe (Windows): Standard Windows installer
  • .msi (Windows): MSI installer for managed deployment
  • .AppImage (Linux): Portable executable - make it executable with chmod +x
  • .deb (Linux): Debian/Ubuntu package
  • .rpm (Linux): Fedora/RHEL package
  • .snap (Linux): Snap package for snap-enabled distributions
  • .dmg (macOS): Disk image installer
  • .zip (Web): Static files ready for deployment

Note: All desktop builds are currently unsigned. You may need to bypass security warnings on first launch.
